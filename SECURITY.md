# Security Policy

## Supported Versions

Only the latest release of TenantShield is supported with security updates. If you find a security issue, please ensure you are running the latest version before submitting a report.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of TenantShield seriously. If you believe you have found a security vulnerability, please do **NOT** report it publicly via GitHub Issues.

Instead, please send an email directly to the contact support listed in your marketplace receipt. 

Please include:
1. A detailed description of the vulnerability.
2. Step-by-step instructions to reproduce the issue (PoC).
3. Any potential impact or exploits.

We will acknowledge your report within 48 hours and work on a remediation plan immediately. Thank you for helping keep our software secure!
